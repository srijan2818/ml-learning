# Machine Learning Algorithms

This repository contains from-scratch implementations of machine learning algorithms written by directly translating mathematical definitions into code.

The purpose is to record and verify understanding, not to optimize performance, provide abstractions, or compete with existing libraries.

## Implemented

### Supervised Learning
- Gaussian Naive Bayes
- Linear Regression (closed form, gradient descent, ridge)
- Logistic Regression (binary, multiclass)
- Perceptron (functional and geometric margins)
- SVM
  - Primal
  - Dual (hard & soft)
  - Kernelized
  - SMO
  - SVR (kernel)
- AdaBoost

### Unsupervised Learning
- PCA
- K-Means
- K-Medoids

## Concepts / Structural Studies
- Spectral Clustering (graph Laplacian, eigen-embedding)
- Gaussian Mixture Model (EM – structural understanding)
- Optimization Layers (GD → Natural Gradient)
